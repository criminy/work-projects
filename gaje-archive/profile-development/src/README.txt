
This is the development release of the gaje-archive system

To run:

	1) Have an existing GAJE database.
	2) Run bin/database-update.sh (TODO, not implemented yet)
	3) Download apache-activemq
	4) Install apache-activemq
	5) Configure bin/setenv.sh
		a) Setup correct locations of activemq inside of setenv.sh
	6) Configure conf/credentials.conf with database and ldap credentials.
